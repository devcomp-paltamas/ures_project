# Ures Project

Modern webes starter alkalmazás gyors terméképítéshez.

## Mit tud most?

- nyilvános landing oldal
- autentikációs alapok
- védett alkalmazásfelület
- mock és valós API működési mód szétválasztása
- magyar és angol nyelvi alap
- világos, sötét és rendszer téma támogatás

## Technológia

- React 19
- TypeScript
- Vite 7
- Tailwind CSS 4
- React Router 7
- TanStack React Query
- Firebase Auth
- Supabase
- Vitest
- Playwright

## Indítás

Telepítés:

```bash
npm install
```

Fejlesztői szerver:

```bash
npm run dev
```

## Fontos parancsok

```bash
npm run lint
npm run typecheck
npm run test:unit
npm run build
```

## Projektcél

Az első kiadás célja egy tiszta, stabil és jól bővíthető alap biztosítása, amely gyorsan továbbfejleszthető valódi termékké.
